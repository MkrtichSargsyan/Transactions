## Getting Started

To get a local copy up and running follow these simple example steps.

### Setup

Instal gems with:

```
yarn install
```

```
bundle install
```

Setup database with:

```
   rails db:drop
   rails db:create
   rails db:migrate
```

### Usage

Start server with:

```
    rails server
```

Open `http://localhost:3000/` in your browser.

### Run tests

```
    rpsec 
```
